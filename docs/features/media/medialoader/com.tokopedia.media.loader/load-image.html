An extension function on ImageView receiver to load an image from some remote url.

This function uses Glide which is a 3rd party library to handle all the networking and caching side of things. Glide handles the loading w.r.t. the lifecycle of the view for you.

Sample Usage

imageView.loadImage("https://someurl.com/someImage")

   setCacheStrategy(MediaCacheStrategy.RESOURCE)
